■ これは何?
YYYY-MM-DD タイプのログファイル(.log)を月ごとにサブフォルダーに移動します
移動先のサブフォルダー名は YYYY-MM を作成します

月に一度、管理権限で実行することを想定します


■ オプション
引数に YYYY-MM を指定すると、指定した年月が処理対象となります
引数を省略した場合は、前月を処理対象となります
